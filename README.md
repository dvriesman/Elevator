# Elevator
Elevate Windows Previleges

It's just a PoC that use Windows API to Elevate running privilege of some application.
In windows 10, it's not enougth to be administrator, you must elevate the privileges.

# Compiling

    ./gradlew build
    
# Executing
    java -jar build/libs/Elevator.jar [PROGRAM TO EXECUTE]
